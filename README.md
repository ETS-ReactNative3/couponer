We're live baby!

Check it out at: https://www.unlimitedcouponer.com/Home

1) cd Couponer
2) npm install
3) cd client
4) npm install
5) cd ..
6) npm run dev
npm run dev will run the client and server at the same time.

Make sure you install the node packages for the server AND the client.
